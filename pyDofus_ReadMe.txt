Quand une nouvelle mise à jour dofus est disponible:

1. Allez dans le dossier du jeu (Dofus\data\common) et copier tout les d2o
2. Coller les d2o dans le dossier input de pyDofus (AnkaBotPC\YayaTools\PyDofus-master\input)
3. Installer python, et éxécuter le fichier d2o_unpack.py (AnkaBotPC\YayaTools\PyDofus-master\d2o_unpack.py)
4. Copier les fichier dans le dossier output de pyDofus (AnkaBotPC\YayaTools\PyDofus-master\output)
5. Coller les fichier dans le dossier D2O des tools (AnkaBotPC\YayaTools\Data\D2O)
